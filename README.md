# Background Color Changer

A simple web application that automatically changes the background color every second when activated.

## Features

- Start/Stop functionality to control the color changing
- Automatic color change every second
- Clean, minimalist interface
- Random color generation

## How to Use

1. Open the HTML file in any modern web browser
2. Click the **Start** button to begin changing the background color
3. Click the **Stop** button to pause the color changes

## Technical Details

The application uses vanilla JavaScript with `setInterval()` to change the background color at regular intervals. Colors are generated randomly using RGB values.

## Files

- `index.html` - Main HTML file containing the structure, styles, and JavaScript

## Browser Compatibility

Works in all modern browsers that support:
- JavaScript ES5+
- CSS3
- DOM manipulation
